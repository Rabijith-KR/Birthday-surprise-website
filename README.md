# 🎂 Happy Birthday Surprise Website ❤️

A beautiful and interactive birthday surprise website created using HTML, CSS, and JavaScript.

This project is designed as a personalized digital birthday gift featuring animations, music, memories, love messages, quizzes, countdowns, and special surprise sections.

## ✨ Features

### 🎉 Interactive Birthday Experience

* Beautiful animated background
* Floating heart animations
* Glassmorphism UI design
* Responsive layout

### 🎵 Background Music

* Play/Pause birthday music
* Seamless looping audio support

### 💌 Personal Letter

* Animated typing effect
* Customizable birthday message

### ⏳ Countdown Timer

* Live countdown functionality
* Days, Hours, Minutes, and Seconds display

### 📸 Memory Gallery

* Display memorable photos
* Lightbox image preview
* Responsive gallery layout

### ❤️ Love Messages

* Interactive love cards
* Personalized messages on click

### 🌹 Relationship Timeline

* Interactive memory timeline
* Clickable memory galleries
* Story-based presentation

### 💕 Love Quiz

* Fun interactive quiz section
* Instant answer feedback

### 🎁 Gift Reveal

* Surprise gift box animation
* Hidden birthday message reveal

### 🎊 Celebration Effects

* Confetti animations
* Final birthday celebration screen

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Canvas Confetti Library
* Google Fonts

## 🎨 Customization

You can easily personalize the website by:

* Replacing `{NAME}` with your loved one's name
* Updating personalized messages
* Adding your own photos
* Replacing the music file
* Editing timeline memories
* Customizing colors and animations

Created to make special moments unforgettable through a personalized and interactive web experience.
